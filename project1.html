Some of my best projects so far

Interfacing Arduino with V-Python

This project focuses on the usage of Arduino Uno board and the Arduino IDE. This project creates a graphical interface, which, for our application, is a car as an example. An ultrasonic sensor would be placed at the back of the car. Whenever an object comes near the ultrasonic sensor, the distance between the object and ultrasonic sensor can be visually viewed on the screen which would be placed at the front and if the object crosses a particular distance limit, a beep sound would be heard which will warn the driver that the obstacle is going to hit the car. The distance of the car from the wall is recorded on the Arduino IDE. It is cost efficient as there is no need of a camera to be fitted.

Loan Default Prediction using Machine Learning

The Loan Default Prediction System is useful for the banks to predict whether a customer will be a Loan Defaulter for their bank or not. It analyzes the previous records of the customers who have taken a loan and then detects whether a person can be a defaulter for the banks from which he would take a loan in later future. The Machine Learning algorithm used for the project is Logistic Regression.

CrowdDynamics: A crowd density estimation approach using GPS mobility data for its dynamics and prediction

Designing a system which will take the commuter’s GPS data of their current location inside the station and thereby get the crowd patterns at the railway station to tackle the problem of over-crowding at railway stations. This data is used for crowd estimation and prediction after certain intervals of time using a machine learning algorithm. The data is visualized using a heat map which is easier to monitor the crowd. This paper discusses on the uses of an interesting Machine Learning algorithm called as “Hidden Markov Model” for crowd prediction.
